# Bugtracker

### Hot to run
1. set up mssql in coldfusion   
2. run `db.sql` locally 
3. open localhost:<port>/src 

### Features
* Log in/Log out
* Bug report creating/showing
  * description
  * statuses
  * history of canges
* List of bug reports
  * showing of bug reports by order of each property
* Updating of user information
